## 🔍 How to Use Kanonarion 📊

Kanonarion functions through a command-line interface. This means you type commands into a small window to tell the tool what to do.

1.  Open the Start menu.
2.  Type "Command Prompt" and press Enter.
3.  Type the location of the tool followed by a command. 
4.  For example, type `kanonarion scan` to look at your current folder.
5.  Wait for the tool to finish the analysis. 

The tool will display a list of information about your dependencies on the screen. It identifies the origin of each library, the license type, and any known security threats.

## 💡 Interpreting the Results 📝

The output provides clear information about your software components:

*   **Dependency Source:** Identifies where the code originates. You see which external libraries support your main application.
*   **License Check:** Shows the legal terms for each library. This helps you track compliance with company policies.
*   **Security Vulnerabilities:** Lists known bugs or security gaps in the code you use. The tool marks items that require your attention.
*   **Reachable Code:** Shows if your application actually sends information to a risky component. This detail helps you focus on real threats rather than minor issues.

## 🤖 Integration for AI Agents 🌐
